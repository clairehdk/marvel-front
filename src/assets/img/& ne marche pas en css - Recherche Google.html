      Pourquoi mon style CSS ne marche pas ?
      La feuille de styles contient une erreur de syntaxe

      Si votre feuille de styles n'est pas valide et notamment si elle contient une erreur de syntaxe importante (même un simple «}» oublié!), le navigateur peut arrêter l'interprétation du code CSS au moment où il rencontre l'erreur. Pensez à utiliser le validateur CSS.      7 janv. 2009
